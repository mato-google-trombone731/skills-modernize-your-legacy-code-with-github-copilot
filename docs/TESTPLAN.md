# Test Plan for COBOL Account Management System

This document outlines test cases to validate the business logic of the existing COBOL application. Use this plan to confirm behavior with stakeholders before implementing equivalent tests in the Node.js version.

The plan covers menu navigation, balance retrieval, crediting, debiting (both successful and failing), invalid input handling, and state persistence.  These cases form the basis for later unit/integration tests in the Node.js application.
